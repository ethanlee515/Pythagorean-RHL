\begin{abstract}
    We propose \emph{Relational Hoare Logic with Pythagorean Errors} in order to apply the cryptographic \emph{Pythagorean probability preservation} property to pairs of probabilistic programs.
    Our program logic allows sharper security analysis by allowing errors to accumulate sublinearly in the size of the considered programs.
    Using our logic then, we verify the security of the \emph{noise flooding} construction for homomorphic encryption schemes using the SSProve \cite{ssprove} framework.
\end{abstract}
